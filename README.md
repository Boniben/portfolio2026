# Portfolio E5 — Benjamin Boniface | BTS SIO SLAM

Site web de présentation développé en PHP natif, hébergé sous XAMPP, à destination du jury de l'épreuve E5 du BTS SIO option SLAM.

Il centralise l'ensemble des situations professionnelles réalisées au cours de la formation, les compétences du référentiel E5 associées, et les documents de preuve correspondants.

---

## Contexte

Je suis moniteur d'atelier en ESAT (Établissement et Service d'Aide par le Travail). En l'absence d'équipe informatique dédiée, j'ai progressivement pris en charge la digitalisation de pratiques internes : outillage métier, gestion de données, sécurisation des accès, communication numérique.

Ce portfolio retrace ces réalisations concrètes et les met en regard du référentiel de compétences BTS SIO.

---

## Projets présentés

- **EasyCleanESAT** — Application métier Symfony de génération de fiches chantier, développée en équipe avec méthode Agile (Scrum).
- **PortfolioBuilder** — Application Java Spring Boot illustrant la mise en place des habilitations utilisateurs via Spring Security (rôles USER / ADMIN, routes sécurisées).
- **TestDriven** — Projet Java démontrant la démarche TDD avec tests unitaires JUnit.
- **Mini-logiciel Excel** — Outil VBA de génération automatique de fiches d'intervention, déployé auprès des utilisateurs de l'ESAT.
- **Communication ESAT** — Participation à la stratégie LinkedIn du directeur, incluant gestion du droit à l'image.
- **Développement de compétences** — Veille technologique, certification SecNumAcadémie, parcours OpenClassrooms.

---

## Stack technique

PHP natif · HTML/CSS · JavaScript · XAMPP

```

documents/         → preuves classées par situation
```
